# 📊 PhonePe Transaction Analysis Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard developed to analyze PhonePe transaction data and generate meaningful business insights through data visualization.

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📈 Dashboard Features
- KPI Cards (Total Transactions, Total Value, Unique Users, Success Rate)
- Transaction Trend Analysis
- Age Segment Contribution
- Service-wise Transaction Analysis
- Weekday vs Weekend Usage
- Interactive Filters (Slicers)

## 💡 Key Insights
- Analyzed over **300K+ transactions**.
- Total transaction value exceeded **₹3.47 Billion**.
- Weekday transactions are significantly higher than weekend transactions.
- Gen X users contributed the highest share of transactions.
